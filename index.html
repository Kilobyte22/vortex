Vortex

The Vortex scripting language, a language that compiles to Lua.

Download this project as a .zip file Download this project as a tar.gz file

Vortex is a scripting language that compiles to Lua. It aims to provide a comfortable, modern scripting language while retaining the simplicity of Lua and its ability to be embedded. The current version is implemented in Lua as well, however, I'm planning to self-host at some point. It's however still very much a work in progress, far from complete.

Convenient

The language uses a curly braced syntax and is inspired by multiple well known languages, including Lua, Scheme, OCaml/F# or Rust.

Multi-paradigm

Vortex features the procedural, functional and object oriented programming paradigms. It inherits its featureset from Lua and adds some more. Unlike Lua, it's a fully expression based language. A short list of new features:

  • Default function arguments
  • Implicit return values
  • Block expressions
  • Builtin coroutine/generator syntax
  • Pattern matching
  • Sequences
  • AST hygienic macros
  • Builtin object syntax
  • Destructuring let
  • Lists
  • Table slicing
  • Table joining
  • and more

Batteries included

Vortex aims to make programming comfortable. Comprehensive standard library is being designed and the language is stuffed with features.

Performance

Vortex-generated code runs as fast as handwritten Lua and is compatible with Lua 5.2 and LuaJIT 2.

Compatibility

You can access any Lua library from Vortex. However, in order to prevent conflicts, Vortex stays in its own environment, accessing Lua through a separate namespace.
